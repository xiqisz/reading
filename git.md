#### git 日常提交

------

1.暂存

```
git add .
```

2.commit

```
git commit -m "modify"
```

3.提交

```
git pull origin master	//上面命令表示，取回 origin/master 分支，再与本地的 brantest 分支合并。
```

4.上传

```
git push origin master
```



#### git 分支管理

------

1.查看分支

```
git branch
```

2.创建分支

```
git branch name
```

3.切换分支

```
git checkout name
```

4.创建并切换分支	2.3合并

```
git checkout -b name
```
